# Promote APIs

## Overview

API promotion allows you to move APIs between environments within your Gravitee Cloud organisation. For example, you can move an API from Dev to QA to Production. This lets you test APIs before moving them to production.

{% hint style="info" %}
For detailed information on API promotion, including supported features, limitations, and step-by-step instructions, see  [APIM API Promotion](https://documentation.gravitee.io/apim/create-and-configure-apis/create-apis/promote-apis)
{% endhint %}
