# Establish private networks with GCP

## Overview

A private network enables a secure connection between your SaaS Gateways and your VPC in your Google Cloud Platform (GCP). A private network establishes a secure interface that allows services to be consumed internally across an isolated VPC, which ensures traffic remains off the public internet while maintaining control.

You can establish a private network in only the following GCP regions:

* `asia-southeast1`
* `europe-west3`&#x20;
* `us-central1`&#x20;

For more information about the Cloud providers and regions that Gravitee supports, see [geography-and-provider-support.md](../../reference/geography-and-provider-support.md "mention").
