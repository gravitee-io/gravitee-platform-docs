# Create a private network

## Overview

You can create a private network between your Gateway and your VPC in the Google Cloud Platform (GCP).

You can establish a private network in only the following GCP regions:

* `asia-southeast1`
* `europe-west3`&#x20;
* `us-central1`&#x20;

For more information about the Cloud providers and regions that Gravitee supports, see [geography-and-provider-support.md](../../reference/geography-and-provider-support.md "mention").

## Prerequisites

* Enable the private network feature. To enable the private network feature, contact your Gravitee representative. For example, your Technical Account Manager.&#x20;
