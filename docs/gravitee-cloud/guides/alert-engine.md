# Alert Engine 

## Overview
Alert Engine is now available for Gravitee Cloud customers. Alert Engine is deployed in the same region as your environments.

### Access Alert Engine 
If Alert Engine is included in your license, you access Alert Engine through your connected API Management (APIM) installations. If Alert Engine is not included in your license, contact your Gravitee repesentative.

For more information about Alert Engine, see [Introduction to Gravitee Alert Engine](../../ae/README.md)

For more information about configuring alerts in APIM, see [Alerts](../../apim/4.11/configure-and-manage-the-platform/gravitee-gateway/alerts.md)
