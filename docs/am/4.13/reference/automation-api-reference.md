# Automation API Reference

## Overview

{% hint style="info" %}
The OpenAPI specs for other versions of AM are available in the documentation for that version.
{% endhint %}

The AM Automation API exposes a machine-oriented HTTP interface for managing Access Management resources declaratively.

* To enable the Automation API, see [Automation API configuration](../getting-started/configuration/configure-automation-api.md).
* For authorization and usage, go to [Automation API guide](https://github.com/gravitee-io/gravitee-access-management/blob/4.12.x/docs/automation/GUIDE.md).
* To access the OpenAPI spec for AM 4.12, go to [OpenAPI spec](https://raw.githubusercontent.com/gravitee-io/gravitee-access-management/4.12.x/docs/automation/openapi.yaml).
