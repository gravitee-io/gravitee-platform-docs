# AI Model Text Classification - Model Reference and Performance Metrics

## Related Changes

Model documentation added for nine AI text classification models across three families: BERT Toxicity (Tiny, Mini, Small), DistilBERT Multilingual Toxicity, Detoxify ONNX, MiniLMv2 Toxic Jigsaw, and Llama Prompt Guard (22M, 86M). Documentation includes model identifiers, HuggingFace repository paths, architecture details, parameter counts, licenses, training datasets, performance metrics (F1, accuracy, precision, recall, AUC-ROC), language support, and selection guidance. Performance tables compare original and optimized (Gravitee) versions for all models. Known limitations and size-accuracy tradeoffs are documented for each model family.
