#### **Native Kafka Endpoint Management**

* Create and configure multiple Kafka endpoints and endpoint groups through the APIM Console UI for Native Kafka APIs.
* Reorder endpoints within groups using drag-and-drop, with the first endpoint in the first group automatically designated as the default.
* Configure bootstrap server addresses and security protocols at the endpoint group level, with optional per-endpoint overrides.
* Load balancer configuration is automatically disabled for Native Kafka APIs to match Kafka's architectural requirements.
