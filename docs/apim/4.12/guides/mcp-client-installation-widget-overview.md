# MCP Client Installation Widget Overview

## Overview

The MCP Client Installation widget (`<gmd-install-mcp>`) enables API publishers to embed one-click installation links and copyable configuration snippets for Model Context Protocol (MCP) servers directly into portal pages. Portal page authors can use this Gravitee Markdown component to provide deep links for supported AI clients (Cursor, VS Code, Claude Desktop) and display transport-specific configuration for HTTP, SSE, and stdio MCP servers. The widget automatically selects the first available installer and provides client tabs for switching between installers.
