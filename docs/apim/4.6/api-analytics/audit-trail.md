## Overview

The Audit Trail is comprised of a table and filtering mechanisms that allow you to audit API consumption and activity. Used in conjunction with dashboard analytics, you can pinpoint an API behavioral change, view the configuration that caused the change, and roll back to an earlier configuration if required.

## Use the Audit Trail

To use the Audit Trail feature:

1. Log in to your API Management Console
2. Select Audits from the left nav
3. Select values for the **Event**, **Type**, and **Enter a date range** filters
4. Click on the eye icon to view patch information
