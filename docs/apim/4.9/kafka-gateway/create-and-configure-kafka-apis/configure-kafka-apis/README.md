# Configure Kafka APIs

## Overview

Kafka APIs can be easily configured using the APIM Console, which is a UI-based tool that allows you to create, modify, and analyze API characteristics and access.

To configure Kafka APIs:

1. Log in to your APIM Console
2. Click on **APIs** in the left nav
3. Select your API

The inner left nav separates API configuration settings into several categories:

<figure><img src="../../../../4.7/.gitbook/assets/A%201%20config%20general%20copy.png" alt=""><figcaption></figcaption></figure>
