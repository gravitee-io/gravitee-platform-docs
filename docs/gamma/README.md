# Gravitee Gamma

Welcome to the Gravitee Gamma documentation. Gamma is a next-generation connectivity platform that unifies API Management, Event Stream Management, Agent Management, Authorization Management, and Platform Management under a shared catalog, authorization engine, and enforcement architecture.

## Product lines

<table data-card-size="large" data-view="cards"><thead><tr><th></th><th></th></tr></thead><tbody><tr><td><strong>API Management</strong></td><td>Govern REST, GraphQL, and gRPC traffic with API proxies, security plans, and policies.</td></tr><tr><td><strong>Event Stream Management</strong></td><td>Manage Kafka clusters, virtual clusters, topics, and event-driven data flows.</td></tr><tr><td><strong>Agent Management</strong></td><td>Govern LLM, MCP, and A2A traffic with the AI Gateway, Catalog, Agent Identity, and Edge Management.</td></tr><tr><td><strong>Authorization Management</strong></td><td>Define and enforce fine-grained GAPL policies across all Gamma traffic types — APIs, MCP servers, AI models, and custom resources.</td></tr><tr><td><strong>Platform Management</strong></td><td>Manage consumer applications and their API subscriptions across the platform.</td></tr></tbody></table>
