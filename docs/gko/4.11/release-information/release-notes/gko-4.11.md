#### **MCP Analytics and Dashboard Template**

* Adds MCP-specific facets and filters for HTTP metrics, enabling dimensional analysis of MCP API traffic by method, tool, resource, and prompt.
* Includes a pre-built MCP dashboard template with 12 widgets covering request volume, latency percentiles, method distribution, and resource/tool/prompt usage.
* Requires Elasticsearch backend with MCP field mappings configured in the analytics index.
* Supports eight widget visualization types including time-series, bar charts, and proportional displays for MCP API monitoring.
