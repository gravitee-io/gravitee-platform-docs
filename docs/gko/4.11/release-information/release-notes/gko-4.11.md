#### **MCP Analytics and Dashboard Support**

* Adds analytics visibility for Model Context Protocol (MCP) proxy traffic, including method invocations, tool calls, resource reads, and prompt requests.
* Introduces four new chart widget types (time-series line/bar, vertical/horizontal bar) and MCP-specific facets and filters for querying analytics data.
* Provides a pre-configured MCP dashboard template with 12 widgets covering request volume, latency percentiles, method distribution, and top-5 rankings for resources, tools, and prompts.
* Requires Elasticsearch backend and MCP proxy metrics collection enabled on target APIs.
